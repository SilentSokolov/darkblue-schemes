# Dark Blue Color Schemes

A color schemes for the [JetBrains PyCharm](http://www.jetbrains.com/pycharm/) - Python and Django IDE **version 2.x**.

# PyCharm v2.x
## Preview

![darkblue-schemes](https://github.com/SilentSokolov/darkblue-schemes/raw/master/darkblue-schemes.png)


## Installation

The XML files should be copied to the following directory:

* MacOS X: `~/Library/Preferences/PyCharm20/colors`
* GNU/Linux: `~/.PyCharm20/config/colors`
* Windows Vista and 7: `C:\Users\$USER\.PyCharm20\config\colors`
* Windows XP: `C:\Documents and Settings\$USER\.PyCharm20\config\colors`
